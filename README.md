# 项目简介

## 核心功能
- 失物信息处理
  - 智能检索：支持多条件组合查询，结合时间筛选
  - 新增失物：支持寻物/拾物模式，包含标题、图片、地点、时间等
  - 编辑更新：实时调整失物状态
  - 审核机制：单条/批量审核，支持通过/拒绝并填写审核意见
  - 特色功能：失物置顶、失物下架、Excel导出、浏览量统计
  - 数据安全：联系方式AES加密存储
- 认领审核处理
  - 完整认领流程：用户申请认领，提交验证信息，管理员审核
  - 特点：认领信息详细展示，批量审核，认领状态全程跟踪，防止冒领
- 失物分类处理
  - 树状结构设计，支持无限级分类，分类图标自定义，排序管理
- 评论处理
  - 管理评论与回复，支持评论内容审核、敏感词过滤、批量删除违规评论
- 数据统计分析
  - 四大核心指标：总发布量、今日发布量、本月找回量、认领成功率
  - 六大可视化图表：失物趋势图、分类统计图、地点热力图、状态分布图、认领统计图、用户活跃度榜
  - 支持按时间范围、失物类型、分类进行多维度筛选

## 适用场景
- 校园环境下的失物招领管理
- 提供失物信息发布、查找、认领功能
- 提高失物招领效率和透明度

## 技术特点
- 前端：Vue 3 + Element Plus
- 后端：Python + MySQL + Redis
- 数据可视化：ECharts
- 技术栈：Vue3 + FastAPI + MySQL + Redis
